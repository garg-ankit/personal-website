# Ankit - Personal Website

Personal website built with [Astro](https://astro.build/) and the React integration.

## Getting Started

This project uses Astro for the site shell and page routing, with React available for interactive UI when needed.

### Prerequisites

- Node.js 20 or newer
- A package manager such as `yarn`

### Install dependencies

Use your preferred package manager from the project root.

```bash
yarn install
```

### Start the Astro dev server

Run the local development server:

```bash
yarn start:dev
```

Astro will start a local server, usually at `http://localhost:4321/`.

### Build for production

Create the production-ready site in `dist/`:

```bash
yarn build
```

### Preview the production build

Serve the built output locally to verify the final production result:

```bash
yarn preview
```

### Useful Astro commands

- `npm run astro -- --help` shows the Astro CLI options
- `npm run astro -- add` adds official Astro integrations
- `npm run astro -- check` runs Astro's project checks
